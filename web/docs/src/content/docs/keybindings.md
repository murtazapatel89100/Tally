---
title: Keybindings
description: TUI keyboard shortcuts for Tally.
---

## View switching

| Key | Action |
|-----|--------|
| `1` or `d` | Switch to Dashboard |
| `2` or `b` | Switch to Balances |
| `3` or `r` | Switch to Register |
| `4` or `a` | Switch to Accounts |
| `Tab` | Cycle to next view |

## Navigation

| Key | Action |
|-----|--------|
| `j` / `↓` | Move down |
| `k` / `↑` | Move up |
| `g` | Jump to top |
| `G` | Jump to bottom |
| `PgDn` | Page down |
| `PgUp` | Page up |
| `Ctrl+D` | Half-page down |
| `Ctrl+U` | Half-page up |
| Mouse wheel | Scroll |
| Left click | Select row |

## Actions

| Key | Action |
|-----|--------|
| `Enter` | Drill into register for selected account |
| `u` / `Esc` | Go back / clear filter |
| `Space` / `o` | Toggle collapse (Balances view) |
| `[` | Collapse all (Balances view) |
| `]` | Expand all (Balances view) |
| `/` | Start filter (shows match count) |
| `n` | New transaction |
| `e` | Edit transaction (Register view only) |
| `?` / `h` | Toggle help overlay |
| `q` / `Q` | Quit |
| `Ctrl+C` | Quit |

## Filter mode

| Key | Action |
|-----|--------|
| _any char_ | Add to filter |
| `Backspace` | Delete last character |
| `Enter` | Confirm filter and stop typing |
| `Esc` | Cancel and clear filter |

## Transaction entry / edit form

| Key | Action |
|-----|--------|
| `Tab` | Next field |
| `Shift+Tab` | Previous field |
| `Space` | Cycle transaction status (in Status field) |
| `↑` / `↓` | Navigate autocomplete list (in Account field) |
| `Enter` | Select autocomplete suggestion |
| `Ctrl+N` | Add a posting row |
| `Ctrl+D` | Delete current posting row |
| `Ctrl+S` | Save transaction |
| `Esc` | Close autocomplete / cancel form |
