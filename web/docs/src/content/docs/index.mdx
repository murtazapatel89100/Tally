## Dashboard

The dashboard gives you a financial snapshot at a glance: a sparkline of net worth over time, a bar chart of top expense categories, and budget gauges for the current month.

![Tally dashboard view showing net-worth sparkline, expense bar chart, and budget progress gauges](/screenshots/ss-dashboard.png)

## Balances

Drill into your account tree. Assets appear in teal, expenses in amber. Collapse and expand any node with `[` / `]`.

![Tally balances view with collapsible account tree](/screenshots/ss-balances.png)

## Register

Browse every posting with running balance. Filter by account or payee with `/`. Amounts are coloured by sign — green for inflows, red for outflows.

![Tally register view with colour-coded amounts and running balance](/screenshots/ss-register.png)

## In-app entry

Press `n` to open the transaction form. Account names autocomplete as you type, and the form validates that postings balance before saving.

![Tally transaction entry form with autocomplete and balance indicator](/screenshots/ss-entry.png)

## Quick start

```sh
# Install from crates.io (once published)
cargo install tally

# Point at your journal
export TALLY_FILE=~/finance/2026.journal

# Open the TUI
tally

# Or run CLI reports
tally bal
tally reg
tally accounts
tally print
```

## Key bindings at a glance

| Key | Action |
|-----|--------|
| `1` / `d` | Dashboard |
| `2` / `b` | Balances |
| `3` / `r` | Register |
| `4` / `a` | Accounts |
| `Tab` | Cycle views |
| `n` | New transaction |
| `e` | Edit selected (Register) |
| `/` | Filter |
| `Enter` | Drill into register |
| `q` | Quit |
